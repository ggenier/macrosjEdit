CEs macros développées pour jEdit permettent de "locker"/"délocker" des fichiers sur un serveur.
Le lockage des fichiers consistent en la création d'un fichier swap "swp".
Ce système est utilisé par vi/vim/gvim et permet d'indiquer aux autres utilisateurs que le fichier est en cours de modification.

Les macros permettent la gestion de ce fichier swp via le plugion FTP de jEdit.
